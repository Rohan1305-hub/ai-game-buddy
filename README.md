# 🎮 AI Game Buddy for Bharat 🇮🇳

An interactive and fun web application built using **Streamlit**, designed to provide simple and engaging mini-games for users. This project demonstrates how AI-inspired interaction and lightweight game logic can be combined into a user-friendly application.

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/ai-game-buddy.git
```

2. Navigate to the project folder:

```
cd ai-game-buddy
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
